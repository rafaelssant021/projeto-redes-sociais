# Projeto Redes Sociais

Acesse o projeto: https://rafaelssant021.github.io/projeto-redes-sociais/

## Sobre
Este projeto simula a navegação em redes sociais dentro da interface de um smartphone, utilizando HTML e CSS.
O objetivo foi recriar a experiência de troca de telas entre diferentes redes sociais através de interações simples e organização de layout.


## Tecnologias utilizadas
- HTML5
- CSS3


## Funcionalidades
- Interface simulando um celular
- Navegação entre páginas usando iframe
- Ícones interativos de redes sociais
- Efeitos visuais com hover
- Layout centralizado e responsivo


## O que eu pratiquei neste projeto
- Uso de iframe para navegação entre páginas
- Posicionamento com CSS (absolute e transform)
- Criação de layouts centralizados
- Aplicação de efeitos visuais com hover e transições
- Organização de múltiplas páginas HTML


## Estrutura do projeto
- index.html → página principal
- páginas internas → telas simuladas das redes sociais
- estilos → arquivos CSS
- imagens → ícones e layout do celular
